

#  Weather App

A responsive weather application that shows real-time weather information using OpenWeatherMap API.

##  Features
- Fetch current temperature, weather conditions, and location
- Search by city
- Responsive design for mobile and desktop

##  Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
